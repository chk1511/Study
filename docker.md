### Docker 
- 특정한 서버 환경에 웹 서비스를 배포하려면 복잡한 설정이 필요하다. 이러한 불편함을 덜어내기 위해 등장한 개념이 '컨테이너'
- 웹 서비스를 배포하는 데에 필요한 소프트웨어를 컨테이너에 담아 구동시킬 수 있다.

### Docker File
- 컨테이너에 담아 구동시킬 소프트웨어를 명시해주고 빌드하면, 이에 맞게 도커 이미지가 생성된다.
